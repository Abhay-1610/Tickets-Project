**Real-Time Support Ticketing** – A full-stack web application enabling customers and agents to manage support tickets with real-time chat using SignalR, built with ASP.NET Core Web API, EF Core (Code-First), MSSQL, JWT authentication, and React + Redux Toolkit.

**Backend**
ASP.NET Core Web API
EF Core Code-First + MSSQL
SignalR - For Live Chat System
JWT Authentication / Refresh Token
Authorization policies +  Strict Role-based access control (Customer/Agent/Admin)
Global exception middleware  / Request logging middleware / Rate limiting

**Frontend**
React
Redux Toolkit
RTK Query
Advanced Redux patterns: entity adapter, memoized selectors, optimistic update, custom middleware, persisted  auth state

**Cloud Storage**
Firebase Storage for ticket images
